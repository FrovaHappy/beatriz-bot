# Beatriz-Bot
Un bot multi-herramientas para mejorar la experiencia de discord

## instalación
```bash
npm install
```
## en desarrollo
```bash
npm run dev
```
## despliegue de slash-command
```bash
npm run dev:deploy
```
## build del proyecto
```bash
npm run build
```
## en producción
```bash
npm run start
```
## Variables de Entorno
```js
DISCORD_TOKEN = ""
DISCORD_CLIENT = ""
DISCORD_GUILD = ""
DEFAULT_COOLDOWN = "un numero entero en segundos"
```
